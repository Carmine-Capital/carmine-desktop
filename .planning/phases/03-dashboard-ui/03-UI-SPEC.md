# Phase 3 — UI Design Contract

> Visual and interaction contract for the Dashboard UI phase. Generated by gsd-ui-researcher, verified by gsd-ui-checker.

---

## Design System

| Property | Value |
|----------|-------|
| Tool | none (vanilla CSS custom properties) |
| Preset | not applicable |
| Component library | none (DOM construction via `document.createElement`) |
| Icon library | Inline SVG (Lucide-style, 15x15 in nav, 14x14 in actions, 24x24 viewBox) |
| Font | Inter variable (`InterVariable.woff2`, local `@font-face`) |

**Source:** Existing `styles.css` token system; CONTEXT.md locked decision "zero new dependencies."

## Spacing Scale

Declared values (multiples of 4, mapped to existing `--space-*` tokens):

| Token | Value | CSS Variable | Usage in this phase |
|-------|-------|-------------|---------------------|
| xs | 4px | `--space-1` | Status dot margin, icon-to-text gap in activity rows |
| sm | 8px | `--space-2` | Drive card internal padding (top/bottom), gap between status dot and drive name |
| md | 12px | `--space-3` | Drive card internal padding (left/right), gap between drive cards |
| lg | 16px | `--space-4` | Section content padding, gap between section heading and first element |
| xl | 22px | (inline `22px`) | Section heading top margin (matches existing `.section-heading` margin-top) |
| 2xl | 28px | (inline `28px`) | Main content padding (matches existing `.main-content` padding) |

Exceptions:
- Status dots are 8px diameter circles (not a spacing token, but a size)
- Auth degradation banner uses `10px 16px` padding (compact alert pattern)
- Drive cards use `12px` padding all around for compact density
- Cache progress bar height is 8px (`--space-2`)

**Source:** Existing `styles.css` spacing tokens `--space-1` through `--space-8`. No new tokens added; all dashboard spacing uses existing values.

## Interaction Contracts

### Panel Navigation

| Behavior | Contract |
|----------|----------|
| Default panel on window open | Dashboard (`activePanel: 'dashboard'`) |
| HTML default state | `class="nav-item active"` on dashboard button, `class="panel active"` on `panel-dashboard` |
| General panel | Loses `active` class, `aria-selected="false"`, `tabindex="-1"` |
| Keyboard nav | ArrowUp/Down cycles through nav items (existing pattern) |

### Real-time Updates

| Behavior | Contract |
|----------|----------|
| Event source | `listen('obs-event', callback)` |
| Setup timing | AFTER initial `Promise.all()` resolves and `setState` is called |
| Update strategy | Incremental patch into `state` (find drive by `driveId`, unshift into arrays) |
| Array caps | `recentActivity` capped at 500, `recentErrors` capped at 100 (match backend buffers) |
| Render debounce | Wrap event-triggered `render()` in `requestAnimationFrame` to batch rapid events |
| Timestamp refresh | 60-second `setInterval` that calls `render()` only when dashboard panel is active. Cleared when panel is not active. |

### Data Loading

| Behavior | Contract |
|----------|----------|
| Initial load | `Promise.all([get_dashboard_status, get_recent_errors, get_activity_feed, get_cache_stats])` alongside existing commands in `init()` |
| State fields | `dashboardStatus`, `recentActivity`, `recentErrors`, `cacheStats`, `activityExpanded` (bool), `writebackExpanded` (bool) |
| Error handling | Wrap in try/catch, call `showStatus(formatError(e), 'error')` on failure |

### Upload Queue Disclosure

| Behavior | Contract |
|----------|----------|
| Default state | Collapsed (`writebackExpanded: false`) |
| Toggle action | `data-action="toggle-writeback-expanded"` toggles `state.writebackExpanded` |
| Animation | Disclosure triangle rotates 90deg via CSS transition (0.15s ease) |
| Content | Per-file name list, one line each |

### Activity Feed Expansion

| Behavior | Contract |
|----------|----------|
| Default state | Show 10 entries (`activityExpanded: false`) |
| Toggle action | `data-action="toggle-activity-expanded"` toggles `state.activityExpanded` |
| Expanded state | Shows up to 500 entries (full buffer) |
| Link copy | Collapsed: "Show all ({N})". Expanded: "Show less" |

### Auth Banner Action

| Behavior | Contract |
|----------|----------|
| Action | `data-action="dashboard-sign-in"` triggers `invoke('sign_out')` then re-auth flow |
| Feedback | Button shows "Signing in..." while in progress via `showStatus()` |

## Helper Functions Required

| Function | Signature | Purpose |
|----------|-----------|---------|
| `formatRelativeTime` | `(isoString: string) => string` | Convert ISO 8601 timestamp to "Just now" / "2m ago" / "3h ago" / "1d ago" / "Never" |
| `formatBytes` | `(bytes: number) => string` | Convert byte count to "2.1 GB" / "512 MB" / "0 B" with 1 decimal |
| `truncatePath` | `(fullPath: string, maxLen: number) => string` | Show `.../{folder}/{file}` from full remote path. Default maxLen 40. |
| `formatSyncStatus` | `(drive: DriveStatus) => string` | Map `syncState` + `uploadQueue` to display text per copywriting contract |
| `aggregateUploadQueue` | `(drives: DriveStatus[]) => {inFlight, queued}` | Sum `uploadQueue.inFlight` and `queueDepth` across all drives |
